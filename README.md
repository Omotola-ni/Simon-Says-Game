
# Simon Says Game

## Description

The Simon Says game is a classic memory and pattern recognition game that tests your memory, logic, and reaction skills. This project is a web-based implementation of the game using HTML, CSS, and JavaScript.

## Installation

1. Clone this repository to your local machine:

   ```
   git clone [(https://github.com/Omotola-ni/Simon-Says-Game.git)]
   ```

2. Open the project folder in your code editor.

3. Open the `index.html` file in your web browser to play the game.

## Usage

1. Start the game by clicking the "Start" button.

2. Simon will display a sequence of colors and sounds.

3. Repeat the sequence by clicking the buttons in the correct order.

4. If you make a mistake and you're in strict mode, the game will start over. Otherwise, you can try again.

5. Complete all 20 rounds to win the game.

## Game Rules

- Simon displays a sequence of colors and sounds.
- The player must repeat the sequence correctly.
- If the player makes a mistake, they can try again.
- In strict mode, a mistake starts the game over.
- Complete all 20 rounds to win.
